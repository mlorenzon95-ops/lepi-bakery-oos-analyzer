# 🥖 L'épi Artisan Bakery - Out of Stock Analyser
> **Identify stock trends and production gaps in seconds.**

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/mlorenzon95-ops/lepi-bakery-oos-analyzer/blob/main/lepi_out_of_stock_analyser.ipynb)

## 🎯 What this tool does for you
This dashboard helps monitor "Out-of-Stock" (OOS) events before 12 PM, helping you adjust daily production based on real trends.

## 🚀 How to use it
1. **Launch:** Click the blue **"Open in Colab"** button above.
2. **Upload:** Click the "Upload" button and select your Excel file.
3. **Analyze:** Choose the month and click **"Run Weekly Analysis"**.
