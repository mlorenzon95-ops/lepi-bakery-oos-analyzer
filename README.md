## 🎯 What this tool does for you
This automated dashboard helps store managers monitor weekly "Out-of-Stock" (OOS) events without manually calculating data. It creates easy-to-read **Heatmaps** that show you exactly which days and products are under-performing.

*   **Save Time:** No more manual Excel sorting.
*   **Visual Data:** Instantly see stock gaps for **Bread**, **Pastry**, and **Deli**.
*   **Better Orders:** Adjust your production based on real historical trends.

## 🚀 3 Steps to your Report
Using this tool is as simple as making a coffee:

1.  **Launch:** Click the blue **"Open in Colab"** button above.
2.  **Upload:** When the program starts, click the "Upload" button and select your Excel file.
3.  **Analyze:** Choose the month from the dropdown menu and click **"Run Weekly Analysis"**.
